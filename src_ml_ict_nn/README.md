## フォルダの構成

```
src_ml_ict_cnn/
        │
        ├── main.py : 実行ファイル.これを実行し、model.pyのCNNクラスを呼び出す.
        ├── module/
        │       ├── activationfun.py : 活性化関数(シグモイド関数)を保存.
        │       └── model.py : DNNのモデルを保存.
        └── README.md : 本ファイル.各種説明記載.

```
